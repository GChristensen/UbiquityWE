Description

UbiquityWE is a browser add-on available by Ctrl+Space key combination that lets you give commands to a browser by entering text into a special input box. Commands are shortcuts for things that you commonly do on the Web; they can eliminate repetitive tasks and let you get more done, faster. Your can create your own UbiquityWE commands in JavaScript.

History

The add-on is a descendant of the original Ubiquity project by Mozilla. Please see the add-on GitHub page for more information about the addon origins.
